# VelocitaElite 🏍️  
**Android Application for Premium Motorcycle Sales**

VelocitaElite is a modern Android application developed as part of a professional training project at **AAST**. The app provides a seamless experience for motorcycle enthusiasts to explore, customize, and inquire about high-end bikes.

## 🚀 Project Overview
This project was refactored and organized to follow **Clean Architecture** principles and the **MVVM** design pattern, ensuring scalability and maintainability.

## ✨ Key Features
* **Showroom Experience:** Browse a curated collection of premium motorcycles with detailed specifications.
* **Bike Configurator:** Interactive screen to customize bike colors and features (e.g., Yamaha FZ6 customization).
* **User Authentication:** Secure Login and Enrollment screens for a personalized experience.
* **Modern UI/UX:** Built entirely using **Jetpack Compose** with a "Luxury Dark" theme.
* **Smooth Navigation:** Robust navigation handling between SplashScreen, Home, and Detail screens.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Asynchronous Tasks:** Kotlin Coroutines
* **Image Loading:** Coil (Remote asset management)

## 📁 Project Structure
The project follows a standard Android package structure:
* `com.velocita.elite.screens`: Contains all UI screens (Login, Showroom, Configurator, etc.)
* `com.velocita.elite.viewmodel`: Holds the business logic and state management.
* `com.velocita.elite.navigation`: Defines the app's navigation graph.
* `com.velocita.elite.ui.theme`: Custom Material 3 styling and color palettes.
